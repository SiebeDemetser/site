Find my car app

Helaas is deze gemaakt voor een oudere android versie. Waardoor ik enkel een screenshot kan toevoegen.
Deze app heeft als bedoeling dat je jouw auto altijd kan terug vinden. Wanneer je parkeert klik je "onthoud mijn huidige locatie"
Als je daarna de weg terug wilt vinden kan je op de knop "toon route" klikken. Deze zal dan via een google maps api de route naar jouw auto tonen.

SpaceInvader

Deze app is om de gyroscope sensor te testen. Je beweegt de raket door naar links en rechts te draaien met de gsm. En schieten is simpelweg door op het scherm te tikken. De ufo gaat met een oneindige loop van links naar recht. De punten zijn heel eenvoudig per hit krijg je een punt.

whack a mole

Deze app heb ik ooit gemaakt in het secundair onderwijs. Verveeling tijdens de les heeft ervoor gezorgt dat ik dit gemaakt heb. De mol verschijnt met een random frequentie op een random plaat op het scherm. De bedoeling is dat je op tijd op de mol klikt voordat hij weg is. De punten zijn ook zeer eenvoudig. Elke hit is een punt.

Carduino

Helaas heb ik hier geen beeldmateriaal meer van. Dit was een zeer typisch carduino projectje. De arduino stuurde dc motors en een servo motor om te sturen. als input had de arduino een HC05 bluetooth module. Voor de app hebben we gebruik gemaakt van MIT app inventor ook wel gekent als Scratch voor android.

Smart alarmsysteem

Het project werkt zoals een modern alarmsysteem. Door middel van een web interface kan je gegevens vragen van de alarmgeschiedenis via smartphone of webbrowser. Je kan ook via de website zonder veel moeite bepaalde sensoren en de configuratie van het systeem aanpassen. Mocht je interesse hebben in de werking kan je een kijkje nemen naar de links onder de afbeeldingen.

aansturing licht en gordijnen

Helaas heb ik hier geen beeldmateriaal meer van. Dit was een eenvoudig python programma dat op een Raspberry pi draait. Deze kon je oproepen via een lokale webpagina. De lichten en gordijnen werden bediend via de gpio pinnen van de Raspberry. Dit project is gemaakt in het secundair onderwijs en is nooit verder gegaan dan een test opstelling.

Body Interaction Game

Dit project was mijn GIP uit 6TSO. De bedoeling was om een game te maken waar je niks moest aanraken om het te spelen. We hebben hiervoor een xbox kinect 360 aangesloten op een windows 7 systeem met unity op geinstalleert. We hebben hiervoor al een website gemaakt. Dus kan je alles over dit project hier terugvinden
